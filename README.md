# Learn Playwright - JavaScript & Playwright Learning Repository

This repository contains learning materials and hands-on examples for Playwright test automation and JavaScript fundamentals.

## Project Structure

### Chapter 1: Basics
- `02_Basics.js` - JavaScript variable declaration basics, naming conventions, and case sensitivity
- `03_VariableBasic.js` - Variable declaration, initialization, and multiple variable assignments
- `Index.html` - Basic HTML file with embedded JavaScript demonstrating `document.write()` and `console.log()`

### Chapter 2: JavaScript Concepts

This chapter covers core JavaScript concepts essential for test automation:

| File | Description |
|------|-------------|
| `const_explained.js` | Demonstrates `const` declarations - constants cannot be reassigned or redeclared |
| `function_explained.js` | Introduction to JavaScript functions - defining and calling reusable blocks of code |
| `Hoisting.js` | Explains variable hoisting with `var` - how declarations are moved to the top during compilation |
| `Hoisting_fun.js` | Demonstrates function-scoped hoisting - variables are hoisted within function scope |
| `javascript-identifier-rules.js` | Comprehensive guide to JavaScript naming rules for variables and identifiers |
| `let_explained.js` | Covers `let` block-scoped variables - cannot be redeclared in the same scope |
| `let_functionscope.js` | Shows block-level scope behavior of `let` inside functions and conditional blocks |
| `var_const_let.js` | Compares `var`, `let`, and `const` - redeclaration, reassignment, and loop scope leakage |
| `var_explained.js` | Explains `var` function-scoped behavior and variable shadowing |
| `var_functionscope.js` | Demonstrates how `var` behaves inside functions vs global scope |

## Key Concepts Covered

- **Variable Declarations**: `var`, `let`, and `const`
- **Scope**: Global scope, function scope, and block scope
- **Hoisting**: How JavaScript engine processes variable declarations
- **Functions**: Creating and calling reusable code blocks
- **Identifier Rules**: Valid naming conventions for JavaScript variables

## How to Run

Execute any JavaScript file using Node.js:

```bash
node chapter_1_Basics/02_Basics.js
node chapter_02_JavascriptConcepts/let_explained.js
```

Or open `chapter_1_Basics/Index.html` in a browser to see DOM manipulation examples.
